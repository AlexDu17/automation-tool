# 二模质量分析工具

读取学校导出的《学生成绩》表格，自动生成一份多 Sheet 的《质量分析结果》表格：等级分布、各科平均分、总分/各科排名分段表、原始成绩明细、分班明细。

## 老师使用说明（Windows，双击运行，不需要装 Python）

1. 找 IT 同事或自己按下面「打包步骤」在一台 Windows 电脑上生成一次 `GradeReportTool.exe`（只需做一次）。生成后如果想要中文名字，直接在文件资源管理器里右键"重命名"改成任何你喜欢的名字（比如"二模质量分析工具.exe"）都可以。
2. 把这个 exe 复制到你自己的电脑上，想放哪个文件夹都可以。
3. 双击这个 exe，会打开一个小窗口：
   - **① 学生成绩表格**：点"浏览…"选择你要分析的成绩表格（`.xlsx` 或 `.xls` 都支持）。
   - **② 分析结果另存为**：默认是"二模质量分析结果.xlsx"，可以直接用，也可以点"另存为…"改名字或改保存位置。
   - 点 **③ 生成分析报告**，等一下就会弹窗提示"分析结果已生成"。
4. 用 Excel 打开生成的文件即可查看以下 Sheet：
   - `等级分布`：每个班每科 A+/A/B+/B/C+/C 的人数和占比（含累计"以上"）。
   - `各科平均分`：每个班每科的均分和班级排名（"教师"一列留空，需要手动填任课教师姓名）。
   - `总分成绩` / `语文成绩` / … / `英语听说成绩`：每科按分数从高到低的分段排名表，每 5 人一段附累计均分。
   - `原始成绩`：全年级按总分排序的明细表，含年级排名。
   - `01`、`02`……：按行政班级分的学生明细表。

### 常见问题

- **提示"找不到输入文件"**：检查文件名有没有打错，或者用"浏览…"重新选一次。
- **提示"无法保存到……"**：一般是这个输出文件正被 Excel 打开着，先关掉 Excel 里的这个文件，或者换个文件名再生成一次。
- **表格里没有化学/某一科**：程序会根据输入表格里实际有的科目自动生成，没有的科目不会出现，属于正常情况。

## 打包步骤（开发者 / IT 同事，只需做一次）

打包 exe 必须在 **Windows** 电脑上进行（不能在 Mac/Linux 上生成 Windows 的 exe）：

1. 安装 Python 3（勾选"Add python.exe to PATH"）：https://www.python.org/downloads/
2. 把整个项目文件夹拷到 Windows 电脑上。
3. 双击运行 `build_windows.bat`，脚本会自动安装依赖并打包。
4. 打包完成后，exe 在 `dist\GradeReportTool.exe`，把它发给老师即可（可以先重命名成中文名字再发）。

> 打包脚本本身只用英文提示信息，是为了避免中文批处理文件在某些 Windows 系统（尤其是非 UTF-8 默认代码页）下出现乱码甚至命令解析错误的问题。

## 开发 / 命令行用法

程序同时支持命令行方式（用于批处理或自己测试）：

```bash
python main.py --input 学生成绩.xlsx --output 二模质量分析结果.xlsx
```

- `--input` / `--output` 可以只写文件名（表示在当前文件夹里找/保存），也可以写完整路径。
- 不带任何参数运行 `python main.py` 会打开图形界面。

### 输入表格格式要求

- 第 1 行标题，第 2 行科目名称（合并单元格），第 3 行"分数/等级"子表头，第 4 行起是学生数据。
- 必须包含"学号""姓名""行政班级"列；行政班级形如两位数字（如 `01`），程序会按"9 + 班级代码"换算成年级班级标签（`01` → 901，代表九年级一班）。
- 科目列由程序自动识别（凡是"分数/等级"成对出现的列都算一个科目），不需要固定是哪几科。
- 如果同时存在"英语""英语听力""英语合并"三列，会自动重命名为"英语笔试""英语听说""英语"（合并后的最终成绩），符合中考英语拆分笔试/听说的惯例；如果没有拆分，则原样使用。

### 项目结构

```
main.py                       # 入口：无参数开图形界面；带参数走命令行
src/grade_report/
  reader.py                   # 读取 .xlsx / .xls（自动识别）
  model.py                    # 表头解析、科目识别、学生数据结构
  analysis.py                 # 等级分布/均分排名/分段表/原始成绩/分班明细 的统计逻辑
  writer.py                   # 写出多 Sheet 的 Excel 结果
  pipeline.py                 # 串联以上步骤，统一转成友好错误提示
  cli.py / gui.py             # 命令行 / 图形界面入口
build_windows.bat             # 一键打包脚本
requirements.txt              # openpyxl / xlrd（运行需要）+ pyinstaller（仅打包需要）
```

### 已知的简化之处

- 输出样式只做基础美化（加粗表头、冻结首行、百分比/两位小数、自动列宽），不追求逐像素还原任何历史模板的颜色边框。
- 各科平均分表的"任课教师"一列无法从成绩表得知，固定留空由老师手动填写。
